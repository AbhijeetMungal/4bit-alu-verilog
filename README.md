A 4-bit Arithmetic Logic Unit (ALU) designed and verified in Verilog HDL.
## Operation supportted
  | Opcode | Operation   | Description |
  |--------|-------------|-------------|
  | 000    | ADD         | A + B       |
  | 001    | SUBTRACT    | A - B       |
  | 010    | AND         | A & B       |
  | 011    | OR          | A OR B      |
  | 100    | XOR         | A ^ B       |
  | 101    | NOT         | ~A          |
  | 110    | Left Shift  | A << 1      |
  | 111    | Right Shift | A >> 1      |
## Tools Used
- ModelSim — simulation and waveform analysis
- GVim — HDL coding
- Cygwin terminal — Git workflow
